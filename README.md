# 💬 Java Multi-Client Chat Application

## 🚀 Project Overview
This is a console-based chat application built using Java Socket Programming and Multithreading.

The application consists of:
- A Server
- Multiple Clients

Multiple clients can connect to the server and communicate in real-time.

---

## 🛠 Technologies Used
- Java
- Socket Programming
- Multithreading
- I/O Streams
